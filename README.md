# openapi-pickler
Simple utility to generate Gherkin specifications from OpenAPI tests.

Run with: 
```bash
$ npm install
$ npm start <url>
$ npm start <filename>
```

## Hosted File
```bash
$ npm start https://petstore.swagger.io/v2/swagger.json
```
## Local File
```bash
$ npm start simpleapi.json
```
Built on the github:gretard/resttestsgenerator library.
